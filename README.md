# Calendari recollida 2026 · versió pro

Projecte Vite + React + PWA per al calendari de recollida domèstica de La Goba, Puigventós i Terrafortuna.

## Novetats de la versió pro

- icona PWA inspirada en el logo que has compartit
- colors verd i rosa ajustats al teu estil visual
- capçalera amb logo
- preparat per publicar a Vercel o Netlify
- installable com a PWA quan estigui en HTTPS

## Instal·lació

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Publicació a Vercel

- puja el projecte a GitHub
- importa el repositori a Vercel
- framework preset: Vite
- build command: npm run build
- output directory: dist

## Publicació a Netlify

- puja el projecte a GitHub
- importa el repositori a Netlify
- build command: npm run build
- publish directory: dist

## Nota sobre el logo

Per restriccions del format d'entrega, he generat una icona SVG inspirada en el logo que has compartit. Si vols que sigui exactament el mateix logo, només has de substituir `public/icon.svg` pel teu arxiu definitiu.
